## EnOS subscription

### 订阅参数来源

- 连接地址 `SUB_HOST`, `SUB_PORT`: 首页-帮助-环境信息-数据订阅;
- 订阅ID `SUB_ID`: 添加数据订阅时自动生成的`订阅ID`;
- 订阅键 `SUB_ACCESS_KEY`: 应用注册之后的`AccessKey`;
- 订阅密钥 `SUB_ACCESS_SECRET`: 应用注册之后的`SecretKey`;

### 操作步骤

- 数据订阅：

> 数据订阅 -> 添加订阅（时序数据订阅/设备模型） ->
> 订阅ID自动生成
> SA账号：选择应用注册时的SUB_ACCESS_KEY
